# Netxxus VPN

Netxxus VPN es un automatizador diseñado para facilitar la conexión en Xuper TV.

## Características

* Automatización de conexión VPN
* Compatible con Xuper TV
* Aplicación ligera y fácil de usar
* Proyecto compartido sin fines de lucro

## Descarga

Descargar la última versión de **Netxxus VPN Automatizador Xuper TV**:

https://github.com/Netxxus/Netxxus-vpn/releases

Versión actual:
**v18**

## Instalación

1. Descargar el archivo APK desde la sección Releases.
2. Instalar en el dispositivo Android o TV.
3. Ejecutar la aplicación.

## Proyecto

Este proyecto se comparte con fines educativos y para facilitar el uso de herramientas en Xuper TV.
